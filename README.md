# Pinecone Hybrid Search

This project demonstrates a **hybrid search system** using **Pinecone** for vector-based semantic search combined with traditional keyword search. It is designed for building **fast, accurate, and context-aware retrieval systems**, often used in **RAG (Retrieval-Augmented Generation) applications**, AI chatbots, and knowledge search engines.

---

## Features

- Combines **vector embeddings** (semantic search) with **keyword matching**.
- Stores and retrieves documents efficiently using **Pinecone vector database**.
- Supports **hybrid ranking** by combining semantic similarity and keyword relevance.
- Easily integrable with **LangChain**, **RAG pipelines**, or other AI tools.
- Ideal for chatbots, enterprise search, and document retrieval.
